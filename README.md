# 🤖 AI Agent–Driven E-Commerce Management System

An AI-powered e-commerce management system that replaces traditional admin dashboards with a natural language chat interface. This project enables administrators to manage store operations using simple conversational commands, improving efficiency and usability.

---

## 🚀 Features

- 💬 **Natural Language Interface**  
  Manage store operations through chat instead of complex admin panels.

- 🛍️ **Product Management**  
  Add, update, and manage products using simple commands.

- 📦 **Inventory Monitoring**  
  Detect low stock levels and receive intelligent restocking suggestions.

- ⏱️ **Task Automation**  
  Schedule recurring tasks using natural language (e.g., weekly reports).

- 🧠 **Context-Aware AI Agent**  
  Maintains short-term conversational memory for better interaction.

- 📊 **Analytics via Queries**  
  Retrieve insights from the database using natural language.

---

## 🛠️ Tech Stack

- **Backend:** Python, Django  
- **AI Integration:** LangChain, Large Language Models (LLMs)  
- **Database:** PostgreSQL  
- **Architecture:** Tool-calling AI agent system
